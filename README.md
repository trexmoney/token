# T. Rex Money contracts

This repo contains contracts related to T. Rex Money.

## Setup
Run following command
```bash
npm i
```

## Compile
Run following command
```bash
npm run compile
```

## List all accounts
```bash
npx hardhat accounts
```

## hardhat cleanup
```bash
npm run clean
```

## Run test
```bash
npm run test
```

## Run Coverage
```bash
npx hardhat coverage
```

##  Deployment
```bash
npx hardhat run --network <Network-name> migrations/1-deploy-token.ts
```
